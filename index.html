Glassdoor Company Metrics Grabber

This tool was designed for Google Chrome Use. It should also work in Safari or Firefox. IT WILL NOT WORK on Internet Explorer.

Set-up

  1. API Key and Partner ID: Sign into Glassdoor and complete this form. You will then receive your API key and Partner ID by email.
  2. Public IP: Visit this link and copy the IP address that's displayed.

Instructions:

  1. Insert API Key, Partner ID, IP and Number of Companies.
  2. Click 'Start'.
  3. Insert Company Names
  4. Click 'Get Glassdoor Data'
  5. Once table appears, Click 'Download Reports'

Note: Companies with generic names may return multiple results
