Glassdoor Company Metrics Grabber

Set-up

  1. API Key and Partner ID: Sign into Glassdoor and complete this form. You will then receive your API key and Partner ID by email.
  2. Public IP: Visit this link and copy the IP address that's displayed.

Instructions:

  1. Insert API Key, Partner ID, IP and Number of Companies.
  2. Click 'Start'.
  3. Insert Company Names and cities (optional)
  4. Click 'Get Glassdoor Data'
  5. Once table appears, Click 'Download Reports'

Usage Notes:

  • This tool was designed for Desktop Google Chrome Use. May not work on Internet Explorer and will not work on mobile.
  • Common names will return multiple matches. Use exact name and city to prioritise correct company.
  • Warning: If using 'Save Videogame Companies Only', note that not all companies have marked the correct industry, so you may miss data.
